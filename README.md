# Sentinel Admin Dashboard (MERN)

A real-time administrative portal built using the MERN stack. This project features a secure login system and an instant notification system for managing student inquiries.

## 🚀 Features
* **Real-time Updates:** Integrated **Socket.io** for instant dashboard notifications without page refreshes.
* **Secure Authentication:** Implemented **JWT (JSON Web Tokens)** for protected admin routes.
* **Vanilla JS Frontend:** Demonstrated deep understanding of the DOM and asynchronous JavaScript.
* **RESTful API:** Built with Node.js and Express to handle CRUD operations.
* **Database:** MongoDB with Mongoose for structured data modeling.

## 🛠️ Tech Stack
* **Frontend:** Vanilla JavaScript, HTML5, CSS3
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Real-time:** Socket.io
* **Authentication:** JWT (JSON Web Tokens)

## 📂 Project Structure
* `/models` - Mongoose schemas (Lead/Inquiry models)
* `/routes` - REST API endpoints
* `/public` - Vanilla JS frontend files
* `server.js` - Main entry point and Socket.io configuration
